## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/harikapadia999/quantumtrade-vr-demo.git

# Navigate to project directory
cd quantumtrade-vr-demo

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:3000` to see the demo.

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

### Deploy to GitHub Pages

```bash
# Deploy to GitHub Pages
npm run deploy
```
